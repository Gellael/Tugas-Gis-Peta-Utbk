# 🗺️ UNIB Smart Navigate - Edisi UTBK

Aplikasi web navigasi cerdas berbasis peta interaktif untuk membantu peserta Ujian Tulis Berbasis Komputer (UTBK) menemukan rute jalan kaki tercepat menuju gedung ujian di lingkungan **Universitas Bengkulu (UNIB)**.

## ✨ Fitur Utama
- **📍 Deteksi Lokasi Otomatis:** Menggunakan GPS perangkat untuk menentukan titik awal pengguna secara *real-time*.
- **🚶‍♂️ Rute Jalan Kaki Terpendek:** Terintegrasi dengan OSRM (Open Source Routing Machine) profil pejalan kaki untuk menghasilkan rute yang akurat melewati jalan setapak kampus.
- **🏢 Database Gedung UTBK:** Pilihan lokasi tujuan yang disesuaikan dengan gedung-gedung utama pelaksanaan UTBK di UNIB (LPTIK, Rektorat, FKIP, dll).
- **💎 UI/UX Modern:** Desain antarmuka *Glassmorphism* yang bersih, responsif untuk perangkat *mobile*, dan ramah pengguna.

## 🛠️ Teknologi yang Digunakan
- **Vite:** Build tool & dev server yang super cepat.
- **Vanilla JavaScript:** Logika utama aplikasi.
- **Leaflet.js:** Library peta interaktif (Open-source).
- **Leaflet Routing Machine:** Plugin untuk kalkulasi dan visualisasi rute.
- **Remix Icon & Google Fonts:** Ikonografi dan tipografi (Plus Jakarta Sans).
